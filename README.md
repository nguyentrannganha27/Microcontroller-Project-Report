# Microcontroller-Project-Report
Hệ thống sử dụng vi điều khiển ATmega128-V2 để giám sát nhiệt độ và khói thông qua cảm biến LM35 và MQ-2. Dữ liệu được xử lý và gửi đến máy tính qua UART. Khi nhiệt độ hoặc khói vượt ngưỡng cài đặt, hệ thống sẽ kích hoạt buzzer, LED cảnh báo và điều khiển quạt nhằm cảnh báo nguy cơ cháy nổ.
